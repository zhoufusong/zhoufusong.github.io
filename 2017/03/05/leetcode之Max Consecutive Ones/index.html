LeetCode-Max Consecutive Ones

问题Max Consecutive Ones

描述：给定二进制的一个数组，找出在数组中连续的1的最大个数，例如[1,1,0,1,1,1]，输出为3

注：

  • 输入数组只包含0和1
  • 输入数组的长度是正整数且不超过10000

算法思路：

一次遍历数组，维护两个变量：当前区间1的个数count和总的1的个数最大值：maxcount，遇到1时count+1，遇到0时count置0，更新count时同时比较count和maxcount的大小来更新maxcount，最后返回maxcount即可
