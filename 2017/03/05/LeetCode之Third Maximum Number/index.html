LeetCode-Third Maximum Number

问题 Third Maximum Number

描述：给定非空整数数组，返回数组中第三大的数，如果它不存在，返回最大的数。时间复杂度必须是O(n)

例子1：

输入：[3,2,1]

输出：1

解释：第三大的数是1

例子2：

输入：[1,2]

输出：2

解释：第三大的数不存在，所以返回最大的2

例子3：

输入：[2,2,3,1]

输出：1

解释：这里的第三大数是指第三大不重复的数，这里的两个2都被认为是第二大数

算法思路：

我们用三个变量first, second, third来分别保存第一大，第二大，和第三大的数，然后我们遍历数组，如果遍历到的数字大于当前第一大的数first，那么三个变量各自错位赋值，如果当前数字大于second，小于first，那么就更新second和third，如果当前数字大于third，小于second，那就只更新third；这里其实用Int.min是有问题的

算法思路：

利用set的自动排序和自动去重复项的特性，很好的解决了问题，对于遍历到的数字，加入set中，重复项就自动去掉了，如果此时set大小大于3个了，那么我们把set的第一个元素去掉，也就是将第四大的数字去掉，那么就可以看出set始终维护的是最大的三个不同的数字，最后遍历结束后，我们看set的大小是否为3，是的话就返回首元素，不是的话就返回尾元素。算法时间复杂度为O(nlgn)
